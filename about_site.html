This site was first created in 2021 after graduating from my academic studies for mobile development. It is created to show all the types of designs and development methods that can go into a website. Creating a website with several designs we can see numerous styles and development techniques. This way of creating a website made it possible to use various concepts, from basic to more advanced web development concepts. Some notable concepts are:
The various "display" attribute methods in CSS, the placing of HTML elements in various positions and locations on the page, adding multi-media components, importing webfonts and web accessibility standards.
